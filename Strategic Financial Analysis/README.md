# Strategic Financial Analysis & Quantitative Modeling

## 📌 Project Overview

This repository contains a comprehensive portfolio of five strategic financial case studies. Each project simulates a real-world financial analyst's environment, requiring rigorous mathematical modeling to support complex decision-making processes.

## 🛠️ Modeling Highlights

The accompanying Excel workbook features advanced quantitative techniques, including:

* **Time Value of Money (TVM):** Discounted Cash Flow (DCF) and Perpetuity models.
* **Fixed Income:** Full amortization scheduling with arbitrage scenario analysis.
* **Capital Budgeting:** Asset Retirement Obligation (ARO) valuation and project IRR/NPV for infrastructure.
* **Modern Portfolio Theory:** Variance-Covariance matrix implementation and risk-return optimization.
* **Derivative Engineering:** Multi-period Binomial Tree for crypto-asset option pricing.

## 📂 Case Study Breakdown

| Case | Title | Core Methodology |
